# Ops.Agent | AI-Enabled Operations Leader

An interactive AI agent that answers questions about a logistics operations candidate's experience, capabilities, and contact information.

## What It Does

- Conversational interface for recruiters and hiring managers
- Pre-built quick actions: Experience, AI Capabilities, Results, Six Sigma, Contact
- Knowledge base covering 8+ years of logistics leadership + AI automation skills
- Zero dependencies—runs entirely in-browser

## Quick Start

1. Download `v1.0` and save as .html file
2. Open in any modern browser
3. Ask questions or click the quick-action buttons

No server required. No API calls. No data collection.

## Tech Stack

- Vanilla HTML/CSS/JavaScript
- Self-contained single file (~15KB)
- Mobile responsive

## Use Case

Built for [Moltbook](https://moltbook.com)—the front page of the agent internet. Demonstrates how operations professionals can leverage AI tools to differentiate themselves in the job market.
